# 什么是节点？

节点简单来讲就是“这一段话”（虽然不太严谨吧），在节点里面可以设置这里要有什么文本、立绘、背景等等，用户每点击一下屏幕就会切换到下一个节点。

比如之前我们写的这个：

```javascript
var 示例 = [
    {
        角色: "Tian",
        内容: "你好，世界。"
    }
];
```

其中：

```javascript
{
    角色: "Tian",
    内容: "你好，世界。"
}
```

就是一个节点。

那么我们在后面再加一个节点：

```javascript
var 示例 = [
    {
        角色: "Tian",
        内容: "你好，世界。"
    },// 注意：这里一定要加英文,号
    {
        角色: "Tian",
        内容: "欢迎使用Dvnge~"
    }
];
```

运行 game.html 再点击一下屏幕就能看到切换对话了。
