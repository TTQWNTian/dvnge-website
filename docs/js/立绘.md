# 立绘

本章介绍怎么使用Dvnge来操作立绘。

## 立绘显示

视觉小说一般都少不了立绘，要在Dvnge设置显示它很简单：

```javascript
立绘: {
    中立绘: { 路径: "assets/img/立绘.png" }
}
```

其中 `中立绘` 可以换成 `左立绘` 或 `右立绘`，也支持一次显示多个：

```javascript
立绘: {
    左立绘: { 路径: "" },
    中立绘: { 路径: "" },
    右立绘: { 路径: "" },
}
```

`立绘` 和 `音乐` 等一样，是会默认延续到下一节点的。

## 立绘切换

```javascript
{
    角色: "Tian",
    内容: "设置立绘",
    立绘: {
        中立绘: { 路径: "assets/img/立绘1.png" },
    },
},
{
    角色: "Tian",
    内容: "默认延续上一节点设置的立绘（继续显示立绘1）"
},
{
    角色: "Tian",
    内容: "切换立绘，直接更换路径即可",
    立绘: {
        中立绘: { 路径: "assets/img/立绘2.png" },
    },
}
```

## 立绘隐藏

```javascript
立绘: {
    中立绘: { 隐藏: true },
},
```

如果说要在同一个节点隐藏一个立绘显示/切换一个立绘就这样写：

```javascript
立绘: {
    右立绘: { 隐藏: true },
    左立绘: { 路径: "assets/img/立绘.png" },
},
```

## 变量路径

这个一般用在玩家有多个选择，而立绘会随着玩家的选择而不同的情况下使用。比如：

![女or男.webp](/docsImg/女or男.webp)

因为选择的不同玩家立绘也会不一样，玩家选择什么开发者是不可预测的。这个时候就需要变量路径，设置方法如下，我们以上面展示的选项为例：

```javascript
{
    内容: "说起来我还没有问你，你是男生还是女生？",
    选项: [
        {
            文本: "我是女生",
            设置变量: {
                "玩家性别": "女"
            }
        },
        {
            文本: "我是男生",
            设置变量: {
                "玩家性别": "男"
            }
        }
    ],
},
{
    立绘: {
        中立绘: { 路径: "主控_{玩家性别}.png" },
    },
},
```
如果选择“我是女生”，那玩家的立绘就会是 `主控_女.png` ，反之则是 `主控_男.png` 。

## 视频立绘

这个一般情况下用不上，不过以防万一有一些特殊需求，引擎还是加入了该功能。
