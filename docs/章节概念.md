# 章节概念

章节是使用Dvnge制作游戏的基本，它们是在 `章节/` 文件夹中的 `js` 文件，我们编写游戏的运行逻辑就是在这些文件中。

**那么该怎么使用呢？**

这里我们在 `章节/` 文件夹中创建一个 `示例.js` 文件,然后里面写上这样的代码：

```javascript
var 示例 = [
    {
        角色: "Tian",
        内容: "你好，世界。"
    }
];
```

之后保存代码文件，这样就写好了一个基础的对话（现在看不懂为什么这么写不要紧，之后都会详细讲解）。

当然只创建文件是不起作用的，我们还需要在 `TTQ视觉小说引擎.js` 中加上所添加的章节路径。

在 `TTQ视觉小说引擎.js` 文件中找到 章节文件列表（大概在第十二行左右），然后在末尾添加这一行的代码：

```javascript
{ 名称: "示例", 路径: "章节/示例.js" }
```

加上后完整的 `章节文件列表` 代码是这样的：

```javascript
const 章节文件列表 = {
    { 名称: "示例", 路径: "章节/示例.js" }
};
```

下载的引擎文件可能还会有“序章”，那个可以暂时先删掉，替换为本教学的“示例”，然后再用搜索替换来把 `TTQ视觉小说引擎.js` 的所有“序章”文本换成“示例”。

这样添加章节的配置就完成了！运行 game.html 就可以看到我们写的对话了。
