# 基础属性

这一章介绍一下Dvnge几个比较简单的基础属性使用。

## 角色

这个在之前的教程中也都有出现过，就是显示在对话框左上角的名字。

![helloworld.webp](/docsImg/helloworld.webp)

*当然我们也可以找到css中的 `.角色` 来改变样式，不过这属于后话了。

在[节点](./什么是节点.md)里面我们这样写角色：

```javascript
角色: "你的角色名称"
```

我们也可以给它加上HTML标签来实现更多特殊效果，比如：

```html
角色: "<span class='red-text'>你的角色名称</span>"
```

然后再在 `css/样式.css` 文件中加上：

```css
.red-text {
  color: #d03838;
}
```

或者我们直接用 `style` 写在HTML标签中：

```html
角色: "<span style='color:red;'>Tian</span>"
```

不过之后如果要复用的话，建议使用 class。

## 内容

`内容` 一般情况下就是这个角色所说的话，在上图显示的白色文字就是 内容。使用代码：

```javascript
内容: "你好，欢迎使用Dvnge。"
```

`内容` 和 `角色` 一样支持HTML标签，可以自由放开想象来尝试实现各种效果~

比如：

```javascript
{
    角色: "Tian",
    内容: `<input type='text' placeholder='输入框'/><button onclick="alert('hello')">按钮</button>`
}
```

运行效果：

![content-input&button.webp](/docsImg/content-input&button.webp)

## 音乐

`音乐` 就是视觉小说的背景音乐，使用起来很简单，写上音乐文件路径就行，如：

```javascript
音乐: "assets/music/背景音乐.mp3"
```

`音乐` 默认循环播放，并且在一个节点设置之后会一直延续下去，要停止 `音乐` 需要在停止音乐的节点把路径改成 `null`，如：

```javascript
{
    内容: "Hello~",
    音乐: "assets/music/背景音乐.mp3"
},
{
    内容: "欢迎使用Dvnge~",
    音乐: null // 在这个节点停止音乐
}
```

## 音效

可以和 `音乐` 一起播放，只在设置音效的节点生效，并且只播放一次。使用方法：

```javascript
音效: "assets/sound/打击.mp3"
```

## 标签

是给节点设置一个方便人理解、记住的标签名，一般在有需要跳转到此对话时使用，设置方法：

```javascript
标签: "标签名称"
```

具体使用示例可查看选项设置。
